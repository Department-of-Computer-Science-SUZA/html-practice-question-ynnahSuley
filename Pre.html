Understanding Body and Pre tags

The body tag in HTML defines the main content section of a web page

The pre tag is used for preformatted text

Here is an example of preformatted text 

        this is a code snippet:
        
        public class MyClass{
            public static void main(String[] args){
                System.out.println("Hello,world!")
            }
        }

As you can see, the code snippetwithin the pre tags maintains its original formatting
